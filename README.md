# Tug

*An IDE for AI coding*

I’m hard at work on Tug. My goal is create an IDE for AI-assistant coding. It’s a personal project *for me*. I’m the target customer. I’m making everything just like I want it. My hope is that the things I choose to please myself might also (someday) please you too.

Creating a new IDE from scratch is a big project, and I’m also building a new component system at the same time, and cool internals like a middleware multiplexed messaging service. So it’s a lot. That said, I’m *very happy* with the daily progress. You can track the work and see all the commits on github.

Before too long, I hope to make a release you can try and enjoy. No schedule though. Things take time.
