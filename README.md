# Tug

An IDE for AI coding

How do we move beyond “vibe coding” with AI and bring more structure to developing complex software with coding assistants?

Tug aims to provide a way, tapping into the incredible facility that AI has with writing code, and backing it up with better planning, architecture, review, committing to produce ready-to-merge pull requests. Tug makes coding with AI feel more like a full-featured software development process.

That's the vision. The project is still in its early phases, and while there's still a lot of work to do, but the workflow and agents in github are now used daily in the process of developing themselves. Exciting times!

Read more at the [website](https://tugtool.dev/) and [journal](https://tugtool.dev/journal/).
