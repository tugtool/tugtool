# Component Normalization

Bring every existing tugways component into full compliance with `tuglaws/component-authoring.md` and L19.

## Approach

One component per dash. Each gets the agent's full attention — read the authoring guide, read the component, bring it into complete alignment. No shortcuts, no "at minimum" passes.

Components are sequenced by pattern similarity. After the first component in each batch is done and approved, it becomes the reference model for the rest of its batch.
